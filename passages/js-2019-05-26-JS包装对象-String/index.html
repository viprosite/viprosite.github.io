JS包装对象-String

String 对象：

  1. 属性：

    length：

  1. 常用方法： let str = ‘abcbcbcbcba’

    String.fromCharCode( UniCode1, …) 返回Unicode码对应字符（串）

    toUpperCase() 全转为大写

    toLowerCase() 全转为小写

    trim() 去除首尾空白字符

    charAt( n ) 返回n处的字符

    charCodeAt( n ) 返回n处的单字符对应Unicode码

    concat( str1, str2…) 返回连接后的字符串

    indexOf( search_str, start) 从start起检索search_str第一次出现的位置索引

    lastIndexOf( search_str, end) 返回end索引(含)前的search_str最后一次出现位置索引

    match(/Reg_exp/) 检索一个或多个正则的匹配，返回匹配的结果数组，没有时返回null

    replace( ‘str’/Reg_exp, new_str) 将匹配的（第一个/全局用正则g）子串替换为new_str

    search(‘str’/Reg_exp) 检索字符串中指定的子字符串str/reg_exp,返回第一次出现位置，没有时返回-1

    slice( start, end) 提取start到end前的字串，负值时从尾部开始 …,-2,-1

    split(‘sep’) 从指定的sep处分割str并返回分割后的多个子串数组

    substr( start, length ) 从start处提取length个字符返回

    substring( from, to ) 返回从from到to前的子串
