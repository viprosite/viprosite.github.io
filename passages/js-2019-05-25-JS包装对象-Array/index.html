JS包装对象-Array

Array对象：

  1. 属性：

    length：元素个数

    constructor：构造函数

  1. 常用方法：

    let arr = [1,2,3]

    改变原数组的:

    push( item… ) 直接向数组尾插入指定的元素并 返回新长度

    pop(): 直接删除最后一个元素并将其返回

    unshift(item…) 直接向数组头插入指定元素并 返回新长度

    shift() 直接删除数组第一个元素并将其返回

    reverse() 直接反转原数组并返回结果

    splice(index, length, item1…) 直接从数组的index处删除length个元素并将剩余参数填入,返回被删元素数组

    sort()

    fill(val, start, end) 直接用val替换数组中从start到end前的元素，end>arr.length时不会添加新元素

    不改变原数组的: let arr = [1,2,3]

    concat([x,x,x…]) 连接多个数组并返回连接后的数组

    filter( (current_val)=>{ return current_val>0}) 返回符合条件的元素数组

    find( (c_val)=>{ return c_val?xxx}) 返回符合条件的第一个元素

    findIndex( (c_val)=>{ return c_val?xxx}) 返回符合条件的第一个元素的位置索引（0开始）

    indexOf( search_element, start_search_index )

    从start处开始检索search_element并返回其第一次出现的位置索引

    includes( search_element,start_search_index )

    从from_index处开始查找是否包含search_element的元素

    some( (c_val, index)=>{ return c_val>0} ) 有一个元素满足条件时返回true不再检测

    map( c_val, index ) 返回每个元素执行回调后的新数组

    forEach( (current_val, index, curent_arr_obj)=>… )：将每个元素传给回调函数处理

    slice( start, end ) 返回start到end前的元素数组

    join( ‘_sep’ ) 用指定连接符将元素连接成字符串返回

    toString() 将 [] 变为 ‘ ‘
